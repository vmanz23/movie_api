Documentation Page for myFlix Movie API

To build the server-side component of a 'movies' web application.
The web application will provide users with access to information about different movies, directors and genres.
Users will be able to sign up, update their personal information,
and create a list of their favorate movies.

Request URL HTTP Method Request Body Response Body
Return a list of ALL movies to the user /movies GET None JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[ID] GET None JSON object holding data about selected movie
Return data about a genre by name/title /genres GET None Returns information about genres selected
Return data about a director by name /directors/[ID] GET None Returns JSON object about selected director
Allow new users to register /users POST JSON object holding data about new user (name, email) JSON object holding data about new user, including UUID
Allow users to update their user info /users/[ID] PUT None Text notifying successful update of user info
Allow users to add a movie to their list of favorites /users/favs POST None Text notifying successful addition to favorites
Allow users to remove a movie from their list of favorites /users/favs DELETE None Text notifying successful removal of favorite
Allow existing users to deregister /users DELETE None Text notifying successful deletion
